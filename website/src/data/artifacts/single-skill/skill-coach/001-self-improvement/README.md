## Lessons Learned

### 1. Progressive Disclosure Works
Lines decreased but value increased. The key: ruthless condensing while preserving essence.

### 2. Anti-Patterns > Templates
Adding "what NOT to do" is more valuable than "what to do." Expert knowledge = knowing the mistakes.

### 3. Focused Iterations
Each iteration had ONE focus:
- Iteration 1: Foundation
- Iteration 2: Actionability
- Iteration 3: Expert Knowledge
- Iteration 4: Usability
- Iteration 5: Decision Support

### 4. Meta-Skills Can Self-Improve
The skill-coach's principles work on the skill-coach itself. This validates the framework.

### 5. Quick Wins Matter
"Add NOT clause to description" takes 30 seconds but has huge impact. Progressive enhancement > big bang rewrite.
