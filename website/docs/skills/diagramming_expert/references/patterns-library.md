## RELATIONSHIP PATTERNS

### One-to-Many

```
┌──────────────────┐            ┌──────────────────┐
│      Parent      │            │      Child       │
├──────────────────┤            ├──────────────────┤
│  id (PK)         │───┐        │  id (PK)         │
│  name            │   │        │  parent_id (FK)  │───┘
│  ...             │   │        │  name            │
└──────────────────┘   │        │  ...             │
                       │        └──────────────────┘
                       │
                       └──────── 1 : N ────────────►
```

### Many-to-Many

```
┌─────────────┐           ┌─────────────┐           ┌─────────────┐
│   Student   │           │ Enrollment  │           │   Course    │
├─────────────┤           ├─────────────┤           ├─────────────┤
│ id (PK)     │◄─────────►│ student_id  │◄─────────►│ id (PK)     │
│ name        │     N     │ course_id   │     N     │ title       │
└─────────────┘     ▲     │ grade       │     ▲     └─────────────┘
                    │     └─────────────┘     │
                    │                         │
                    └──── N : N ──────────────┘
                         (via junction table)
```
