## Retirement Mathematics

### Sequence of Returns Risk

Two portfolios with same AVERAGE return can have wildly different outcomes depending on order of returns. Early losses + withdrawals = permanent damage.

### Safe Withdrawal Rate by Valuation

```
CAPE < 12: 5.0%+ historically safe
CAPE 12-18: 4.0% historically safe
CAPE 18-25: 3.5% more prudent
CAPE > 25: 3.0-3.5% recommended
```
