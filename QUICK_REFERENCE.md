# Quick Reference Guide

## One-Minute Quick Start

**Want everything?** Use the Orchestrator:
```
I need [describe your project]. Coordinate all specialists to deliver 
everything: research, design, design system, and team plan.
```

**Want something specific?** Use the right specialist:

| Need | Use This Skill | Example Prompt |
|------|----------------|----------------|
| Brand identity | Web Design Expert | "Create a brand identity for [app] targeting [audience]" |
| Design documentation | Design System Creator | "Create a design bible with CSS for [project]" |
| Research insights | Research Analyst | "Research best practices for [topic]" |
| Team planning | Team Builder | "Design a team to build [project type]" |
