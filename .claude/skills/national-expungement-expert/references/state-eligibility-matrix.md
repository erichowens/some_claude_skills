# State Eligibility Matrix

Quick reference for expungement eligibility across all 50 states + DC.

## Legend

**Eligibility Rating:**
- A = Excellent (most offenses, short waits)
- B = Good (many offenses, moderate waits)
- C = Fair (limited offenses, longer waits)
- D = Poor (few offenses, very long waits)
- F = Very Limited (almost nothing qualifies)

**Clean Slate:**
- Y = Has automatic expungement
- P = Pending/partial automatic
- N = No automatic expungement

## Detailed State Profiles

### A-Rated States (Excellent)

#### California (CA)
```
Clean Slate: Yes (AB 1076, 2020)
Terminology: Dismissal (PC 1203.4)
Misdemeanor wait: 1-2 years from completion
Felony wait: 1-2 years from completion (non-violent)
Key exclusions: Sex offenses, violent felonies
Special: Marijuana automatic (AB 1793)
```

#### Connecticut (CT)
```
Clean Slate: Yes (PA 21-32, 2021)
Terminology: Erasure
Misdemeanor wait: 7 years (automatic)
Felony wait: 10 years Class D/E (automatic)
Key exclusions: Class A-C felonies, sex offenses
Special: One of strongest Clean Slate laws
```

#### Illinois (IL)
```
Clean Slate: Yes (SB 2122, 2024)
Terminology: Expungement/Sealing
Misdemeanor wait: 2-3 years
Felony wait: 3-5 years (sealing, not expungement)
Key exclusions: Violent crimes, sex offenses, DUI
Special: Extensive automatic provisions
```

#### Michigan (MI)
```
Clean Slate: Yes (2020)
Terminology: Set aside
Misdemeanor wait: 7 years (automatic)
Felony wait: 7-10 years
Key exclusions: Assaultive crimes, crimes > 10 years
Special: First state with felony Clean Slate
```

#### Minnesota (MN)
```
Clean Slate: Yes (2023)
Terminology: Expungement
Misdemeanor wait: 2-5 years
Felony wait: 5-7 years
Key exclusions: Murder, sex offenses
Special: Automatic after waiting period
```

#### New Jersey (NJ)
```
Clean Slate: Yes (2019)
Terminology: Expungement
Misdemeanor wait: 5 years (can be immediate if only one)
Felony wait: 6-10 years
Key exclusions: Murder, robbery, sex offenses
Special: "Clean Slate" law model for others
```

#### New York (NY)
```
Clean Slate: Yes (2023)
Terminology: Sealing
Misdemeanor wait: 3 years (automatic)
Felony wait: 10 years Class D/E (automatic)
Key exclusions: Sex offenses, Class A-C felonies
Special: Effective 2027 for felonies
```

#### Oregon (OR)
```
Clean Slate: Yes (2025)
Terminology: Set-aside
Misdemeanor wait: 3 years from completion
Felony wait: 7 years (Class C), 10 years (person felonies)
Key exclusions: Class A felonies, sex offenses, DUI
Special: Marijuana only 1 year
```

#### Pennsylvania (PA)
```
Clean Slate: Yes (2018 - first in US)
Terminology: Expungement/Sealing
Misdemeanor wait: 10 years arrest-free (automatic)
Felony wait: Petition only, varies
Key exclusions: Violent crimes, multiple convictions
Special: First Clean Slate state
```

## Special Categories

### Marijuana-Specific Relief

States with enhanced marijuana expungement:
- **California**: Automatic review (AB 1793)
- **Illinois**: Automatic < 30g possession
- **Colorado**: Cannabis conviction sealing
- **New Jersey**: Included in Clean Slate
- **Oregon**: 1-year wait (vs 3 for other misdemeanors)
- **New York**: Automatic for marijuana convictions
- **Vermont**: Automatic for possession < 1 oz

### Juvenile Records

Most states seal juvenile records automatically:
- **California**: Sealed at 18, destroyed at 21
- **New York**: Auto-sealed at 21
- **Texas**: Sealed at 21 for most
- **Illinois**: Sealed upon reaching 17+
- **Michigan**: Sealed at 18 for most

Exceptions requiring petition:
- Sex offenses involving minors
- Violent felonies tried as adult
- Cases transferred to adult court

### Diversion/Deferred Adjudication

States where diversion = easy expungement:
- **Oregon**: Dismissed upon completion
- **California**: Diversion completion = dismissal
- **New York**: ACD (Adjournment in Contemplation of Dismissal)

States where deferred = still conviction:
- **Texas**: Deferred adjudication still appears
- **Georgia**: First offender still conviction for some purposes
