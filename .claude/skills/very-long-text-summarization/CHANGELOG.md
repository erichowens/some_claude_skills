# Changelog: very-long-text-summarization

## v1.0.0 (2026-02-05)

### Created
- Three-pass hierarchical extraction (Haiku army → Sonnet synthesis → Opus refinement)
- Structured knowledge map output (concepts, processes, expertise patterns, temporal evolution, metaphors)
- Three output modes: summary, knowledge-map, skill-draft
- Semantic chunking with overlap
- Cost model (~$0.19 for a 300-page handbook)
- Anti-patterns: single-pass, unstructured output, no overlap, no traceability
- Designed for processing professional handbooks and biographies for skill extraction
